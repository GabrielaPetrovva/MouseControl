# Camera Mouse Control (Hand Gesture Controller)

Софтуер за управление на мишката чрез жестове с ръка в реално време, използвайки MediaPipe, OpenCV и PyAutoGUI.

---

## Описание

Този проект позволява управление на компютъра чрез камера, без използване на физическа мишка. Системата разпознава жестове на ръката и ги преобразува в системни действия.

---

## Функционалности

- Движение на курсора чрез показалец
- Ляв клик чрез събиране на палец и показалец (пинч жест)
- Двоен клик чрез тройна щипка (палец + показалец + среден пръст)
- Drag & Drop чрез задържан пинч (>0.55 сек)
- Десен клик чрез три пръста (показалец + среден + безименен)
- **Начална страница (Home Screen)** с преглед на всички режими
- **Puzzle режим** с интерактивни изображения и 9 теми
- **Tutorial режим** с интерактивно обучение стъпка по стъпка
- **Презентация режим** с жест-базирана навигация между слайдове
- **Рисуване режим** с пръст във въздуха
- **Десктоп режим** – скрива прозореца, жестовете продължават да работят
- **Настройки панел** с навигация чрез стрелки

---

## Жестове

| Жест | Действие |
|---|---|
| Показалец (само) | Движение на курсора |
| Пинч (палец + показалец) | Ляв клик |
| Задържан пинч (>0.55 сек) | Drag & Drop |
| Тройна щипка (палец + показалец + среден) | Двоен клик |
| Показалец + среден + безименен | Десен клик |
| 5 пръста, задържани 1.5 сек | Отваря Home меню |
| 4 пръста (без палец), задържани 1.5 сек | Отваря Настройки |

> **Тройна щипка:** трите пръста (палец, показалец, среден) се събират близо един до друг. Останалите пръсти могат да са в произволно положение — свити или изправени.

## Режими

### Начална страница (Home Screen)

При стартиране автоматично се появява начална страница с всички налични режими. Всеки режим е представен с клавишен shortcut, описание и визуална икона.

Може да се отвори по всяко време чрез задържане на отворена длан (5 пръста) за 1.5 секунди, или с клавиш `H`.

| Клавиш | Действие |
|---|---|
| H | Отвори / затвори начална страница |
| ESC | Затвори начална страница |

За избор на режим от менюто — задръжте ръката върху реда 1 секунда (dwell), или натиснете съответния клавиш.

---

### Tutorial Mode (Урок)

Интерактивно обучение с 4 стъпки:

1. Движение на курсора
2. Ляв клик (пинч)
3. Десен клик
4. Запознаване с Puzzle режима

Всяка стъпка изисква задържане на правилния жест за кратко време. Прогресът се визуализира с лента в долната част на панела.

При влизане в Tutorial, Puzzle, Презентация или Рисуване режим се появява модален въпрос:

> **Искаш ли да видиш обучението?**
> - `Y` – Стартира урок режим преди влизане
> - `N` – Влиза директно в избрания режим
> - `ESC` – Отказ

---

### Puzzle Mode (Пъзел)

Нареждане на пъзел от изображения с 9 теми: Океан, Залез, Гора, Космос, Бонбони, Планини, Абстракт, Плаж, Лого.

Изображенията се зареждат от папка `puzzle_images/` с contain логика (запазен aspect ratio). Поддържа PNG с прозрачност (алфа канал). При липсващ файл се генерира процедурен градиент.

Управлението е чрез пинч жест — хващане и пускане на парчета. Задръжте курсора върху парче 2 секунди за подсказка (призрачна целева позиция).

Рекордите се записват автоматично в `puzzle_records.json`.

#### Контроли в Puzzle Mode

| Клавиш | Действие |
|---|---|
| P | Включване / изключване |
| N | Следваща тема |
| B | Предишна тема |
| R | Разбъркване |
| M | Визуално меню с теми (избор чрез dwell 1.2 сек) |
| Enter | Потвърждаване на тема от менюто |
| D | Смяна на трудност (2×2 / 3×3 / 4×4) |

---

### Презентация режим

Управлява слайдове в PowerPoint, Impress или друга презентационна програма. Програмата автоматично открива и фокусира презентационния прозорец.

| Жест | Действие |
|---|---|
| Пинч | Следващ слайд |
| 3 пръста (десен клик жест) | Предишен слайд |
| Свайп наляво | Следващ слайд (fallback) |
| Свайп надясно | Предишен слайд (fallback) |

При промяна на слайд се показва визуална обратна връзка — цветен flash от съответната страна и голяма стрелка с надпис в центъра.

---

### Рисуване режим

Рисуване с показалеца директно върху видео кадъра.

| Жест / Клавиш | Действие |
|---|---|
| Показалец (само) | Рисуване |
| Пинч | Смяна на цвят |
| Z | Смяна на дебелина на линията |
| C | Изчистване на платното |
| W | Изход от режима |

Налични цветове: зелено, синьо, циан, оранжево, бяло, розово, жълто.
Налични дебелини: 2px, 4px, 7px, 12px.

---

### Десктоп режим

Скрива прозореца на програмата изцяло — камерата продължава да следи ръката и всички жестове работят нормално. Позволява свободна работа на десктопа без прозорецът да пречи.

За активиране: клавиш `D` от Home менюто или главния прозорец.
За изход: натиснете `D` отново.

Работи на Windows (чрез WinAPI), Linux (чрез xdotool) и macOS (чрез AppleScript).

---

### Настройки

Панел за настройка на параметрите в реално време.

| Параметър | Описание |
|---|---|
| Плавност (smooth) | Брой кадри за изглаждане на движението |
| Разст. клик | Разстояние между пръстите за регистриране на пинч |
| Cooldown клик | Минимално време между два клика |
| Drag задържане | Колко дълго трябва да се задържи пинч за Drag |

| Клавиш | Действие |
|---|---|
| ↑ / ↓ | Избор на параметър |
| ← / → | Промяна на стойността |
| S / ESC | Затваряне |

> Настройките използват `cv2.waitKeyEx()` за правилно разпознаване на стрелките на Windows, Linux и macOS.

## Инсталация

### Клониране

```
git clone https://github.com/your-username/camera-mouse-control.git
cd camera-mouse-control
```

### Виртуална среда

Windows:
```
python -m venv venv
venv\Scripts\activate
```

Linux / Mac:
```
python -m venv venv
source venv/bin/activate
```

### Зависимости

```
pip install opencv-python mediapipe pyautogui numpy pillow requests
```

> **Десктоп режим на Windows:** за допълнителен fallback може да инсталирате `pygetwindow`:
> ```
> pip install pygetwindow
> ```

---

## Стартиране

```
python mouse_control.py
```

MediaPipe моделът (`hand_landmarker.task`) се изтегля автоматично при първо стартиране. При стартиране се появява начална страница с всички налични режими и техните клавишни shortcuts.
